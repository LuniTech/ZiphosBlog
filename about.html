Why Did I Create This Blog?

A friend of mine once asked whether I had an online blog, and that's where most of my inspiration came from. Some of it also stemmed from the exciting prospect of accessing my thoughts and experiences from any location. Creating a blog allowed me to enhance the entries with features like pictures, audio, and links, making the whole process more engaging. I could have easily created a pre-made blogger account, but I decided to work through a tutorial from scratch. Why, you might wonder? It was a valuable opportunity for me to engage in instructive and educational experiences, ultimately improving my skills and insights as a MERN stack developer. I've always found DIY projects to be more satisfying than simply buying ready-made products. Of course, the onus is on me to actively learn and understand the code rather than just copying and pasting it. Undoubtedly, the modifications I made serve as evidence that I understand the code and can easily make improvements and adjustments to it.

Technical Information and Technologies Used to Create This Blog

  • Frontend: The frontend of this blog is built using basic HTML, CSS, and JavaScript, with some Bootstrap for styling buttons and other elements.
  • Backend: On the backend, I've used Node.js, combined with MongoDB and Mongoose, to store the entries and perform CRUD operations.
  • AWS Services: I've integrated some AWS services like Polly and S3 to create audio versions of some of the newer entries. It's important to note that AWS services may require payment for continued use after a year.
  • I avoided using AWS's polly as API for security reasons and to avoid extra costs
